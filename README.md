# Data Cleaning & Exploratory Data Analysis Project

This project focuses on cleaning and analyzing two datasets:
1. Netflix Movies & TV Shows
2. E-commerce Transactions

## Objectives
- Clean and prepare raw datasets
- Perform exploratory data analysis (EDA)
- Identify key trends and insights

## Key Insights

### Netflix
- Movies dominate the platform
- Content increased rapidly after 2015
- Most content is rated TV-MA

### E-commerce
- A few products drive most sales
- UK generates the highest revenue
- Customer spending is highly concentrated

## Tools Used
- Python (Pandas)
- Jupyter Notebook
- Matplotlib / Seaborn
